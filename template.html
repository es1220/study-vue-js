디렉티브

  • v- 접두사가 있는 특수 속성. 속성값은 단일 Javascript 표현식 (단 v-for 예외)
  • v-if : 속성값에 따라 엘리먼트를 제거 또는 삽입
  • v-bind(:) : 전달인자(예 - `:href`)를 사용하여 반응적으로 해당 속성 갱신
  • v-on(@) : DOM 이벤트를 수신. 전달인자는 이벤트를 받을 이름.
